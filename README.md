Desenvolvi um e-commerce completo com foco em experiência do usuário e integração com backend, permitindo a navegação de produtos, gerenciamento de carrinho e finalização de compras de forma funcional.

O sistema conta com funcionalidades como listagem e filtragem de produtos, carrinho persistente utilizando localStorage, fluxo de checkout em múltiplas etapas, cálculo automático de valores (subtotal, frete e total) e histórico de pedidos por usuário.

Implementei integração com o Supabase para armazenamento e gerenciamento de dados, incluindo operações de CRUD para produtos e pedidos, além de controle de usuários e persistência de informações no banco de dados.

O projeto também possui um painel administrativo para gestão de pedidos e produtos, organização modular do código em JavaScript e uma página de diagnóstico para testes de integração e validação do sistema.

Tecnologias utilizadas: JavaScript, HTML5, CSS3, Supabase e Bootstrap.
